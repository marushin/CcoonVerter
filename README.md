## 🚀 CcooNverter v0.8.5 공식 출시

강력하고 빠른 RAW 이미지 배치 컨버터, **CcooNverter**의 첫 번째 공식 버전을 공개합니다! 
사진 작가의 워크플로우를 최적화하기 위해 설계된 다양한 기능들을 만나보세요.

### ✨ 주요 업데이트 및 기능 (Key Features)

- **🎨 12가지 픽처 스타일 지원**: Standard, Portrait, Vivid부터 Cinematic, HDR까지 다양한 스타일을 한 번의 클릭으로 적용 가능합니다.
- **⚙️ 세밀한 톤 컨트롤**: 밝기뿐만 아니라 RGB 채널 조정, Black/White Point 설정을 통해 이미지의 깊이를 제어할 수 있습니다.
- **📸 고품질 RAW 프로세싱**: Sony(.ARW), Canon(.CR2, .CR3), Nikon(.NEF), Adobe(.DNG) 등 주요 RAW 포맷을 완벽하게 지원합니다.
- **💾 프리셋(Preset) 시스템**: 나만의 설정값을 프리셋으로 저장하고 언제든 다시 불러올 수 있습니다.
- **⚡ 멀티프로세싱 최적화**: 멀티코어 CPU를 활용하여 대량의 사진을 매우 빠른 속도로 화질 손상 없이 변환합니다.
- **🖼️ EXIF 보존**: 이미지의 촬영 정보를 가능한 만큼 변환합니다.

### 🛠️ 설치 및 사용 방법
- 아래 `CcooNverter.exe`를 다운로드하여 바로 실행하세요. (설치가 필요 없는 포터블 앱입니다.)
- Windows 환경에 최적화되어 있습니다.
